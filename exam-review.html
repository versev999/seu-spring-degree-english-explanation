I. 考试结构与复习路径

客观题 50 分

单选、判断、配对、填空、摘要/结论排序。主要考概念、规则和识别。

主观题 50 分

改写、学术汉译英、图表作文、摘要写作或论文模块写作。主要考表达输出。

考前不要平均用力。优先顺序建议为：学术风格规则 → 名词化/去口语改写 → 图表作文/摘要写作 → 引用、参考文献、投稿信判断 → 各单元细节。
题型最容易考什么复习抓手
单选 / 判断学术风格、参考文献、摘要、结论、图表类型、时态、强弱语气先背规则，再看关键词
配对文体类型、论文板块、引用类型、strengthening / weakening抓句子功能，不按单词硬猜
排序摘要或结论段落逻辑目的 → 方法/对象 → 结果 → 意义/局限

II. 客观题速训

A. Academic Style 判断

学术写作四个核心特征：objectivity（客观）、complexity（句式和信息密度）、formal structure（正式结构）、reference（规范引用，避免 plagiarism）。

In academic writing, directly addressing the reader as "you" is usually preferred.

通常应避免直接称呼读者为 you，可改为被动语态、形式主语或无人称表达，使语气更客观。

A conclusion should not introduce a completely new idea at the end of the paper.

结论用于总结、评价、指出局限和未来方向，不应突然加入正文没有论证的新观点。

B. Citation Type 配对

Drill Match each citation with its type.

SentenceType
Smith (2020) argues that online feedback improves learner autonomy.
Online feedback may improve learner autonomy (Smith, 2020).
Several authors have reported that online feedback supports learner autonomy (Smith, 2020; Wang, 2021).
作者做主语是 author prominent；作者只在括号里是 information prominent；Several authors / researchers 这类概括式表达属于 weak author prominent。

C. 强化与弱化语气

Which expression weakens a claim?

may / might / could / appear / seem / likely / possible 等表示谨慎推断，属于 weakening 或 hedging。

Which expression strengthens a claim?

strong support, clearly, undoubtedly, confirm, establish, prove 等会加强论断。

D. 摘要 / 结论排序

排序题优先按功能判断：研究目的 → 研究对象/材料 → 方法 → 结果 → 结论/启示。如果是结论段，常见顺序为：回到研究目标 → 总结发现 → 说明意义 → 承认局限 → 提出后续研究。

III. 改写与翻译

A. Nominalization 名词化

名词化就是把动词或形容词变成名词，让句子更正式、更抽象、更适合学术写作。考试中常考“动词变名词 + 句式重构”。
Verb / adjectiveNoun formExample pattern
allowallowancethe allowance of...
proposeproposala research proposal
removeremovalthe removal of informal expressions
refuserefusalthe refusal to...
indicateindicationan indication of...
assumeassumptionthe assumption that...
intendintentionthe intention to...
negligentnegligenceevidence of negligence
legallegalitythe legality of...
proficientproficiencyEnglish proficiency

Rewrite Turn actions or qualities into noun phrases.

  1. Crime was increasing rapidly and the police were becoming concerned.
  2. Students used the library extensively, so more resources were needed.
显示参考改写
  1. The rapid increase in crime caused concern among the police.
  2. The extensive use of the library indicated a need for more resources.

B. 去口语化 Academic Voice

Informal / spokenAcademic replacementRule
try to find outdiscover / investigate短语动词换正式动词
look at carefullyexamine用单个精确动词
make sure ... is trueverify动词更准确
showdemonstrate / indicate按语气选择
get rid ofeliminate / remove避免口语短语
a lot ofnumerous / a large number of避免 vague words
bigsignificant / substantial用更精确的形容词
endconclusion使用学术名词
高频规则：不用缩写（won't → will not）；不用直接问句；限制 etc. / and so forth；避免 I / we / you；必要时使用被动语态。

IV. 论文模块模板

Introduction

  1. Background: In recent years, there has been a dramatic increase in...
  2. Literature Review: There is a general consensus that...
  3. Gap: However, few studies have focused on...
  4. Your Paper: The primary objective of this paper is to...

Methodology

  1. Objective: The study aims to examine...
  2. Subjects: A group of ... were recruited as participants.
  3. Procedure: Copies of the questionnaire were given to...
  4. Data treatment: The data were computer-analyzed using SPSS.

Results & Discussion

  1. Summary: One key issue in this study was...
  2. Result: The preliminary analysis discovered that...
  3. Discussion: Consistent with previous studies, we observed...
  4. Implication: This study has several implications for...

Conclusion

  1. Restate the research objective.
  2. Summarize the main findings.
  3. Explain implications.
  4. State limitations and future research.

Abstract

  1. Background / Objective: present tense
  2. Methods: past tense
  3. Results: past tense
  4. Conclusion / Significance: present tense

References & Submission

  1. References must include all cited sources.
  2. Paraphrased materials also need citations.
  3. A cover letter states title, originality, journal fit, conflicts, and contact information.

V. 写作高分模块

A. 图表作文 Graph Description

图表作文最稳结构：介绍图表 → 描述关键数据和趋势 → 总结整体变化。不要把每个数字都写出来；优先写 peak、bottom、fluctuation、overall trend。
功能表达
介绍图表The line graph / bar chart / table shows...
上升a slight / gradual / steady / rapid increase; an upward trend
下降a marked / dramatic / sharp decrease; a downward trend
波动fluctuate between ... and ...
最高/最低reach a peak of... / bottom out at...
对比whereas, while, by contrast, compared with, higher than

Template Three-sentence graph paragraph

显示模板
The chart shows changes in [topic] from [time A] to [time B]. Overall, there was [an upward / a downward / a fluctuating] trend, with the figure reaching its peak at [number] in [time]. The most noticeable change occurred between [time] and [time], when [data] increased / decreased from [number] to [number].

B. 摘要写作 Abstract Writing

一句话顺序：Background + Objective + Method + Result + Conclusion / Implication.
例：This study investigates... By using..., the data were analyzed... Results showed that... These findings suggest that...

C. 投稿信与回复信

Submission Cover Letter

  • manuscript title
  • article type and target journal
  • originality statement
  • no conflict of interest
  • main contribution
  • corresponding author information

Reply to Reviewers

  • Thank the editor and reviewers.
  • State that revisions have been made.
  • Respond point by point.
  • Use polite expressions: Thank you for raising this point.
  • Identify where the manuscript was changed.
